Collections

It is the root interface of the collections framework in Java (except for Map). It defines the general methods that any collection must implement. It groups objects in a single place and provides basic operations such as adding, removing, and verifying elements.

It does not have a direct implementation but is extended by other interfaces such as List, Set, and Queue.

Key methods:

  • add(E element): Adds an element to the collection.
  • remove(Object o): Removes an element from the collection.
  • contains(Object o): Checks if an element is present in the collection.
  • size(): Returns the number of elements in the collection.
  • clear(): Removes all elements from the collection.
  • isEmpty(): Checks if the collection is empty.
  • iterator(): Returns an Iterator to traverse the elements.

The following classes inherit from this interface:

  1. List
  2. Set

An ordered collection that allows duplicate elements. The elements are indexed and can be accessed through their position.

Base interface: Inherits from Collection.

Main implementations:

  • ArrayList: Based on dynamic arrays. Fast access by index, but insertions/deletions are costly.
  • LinkedList: Based on a doubly linked list. Better for frequent insertions and deletions.

Key methods:

  • get(int index): Returns the element at the specified position.
  • set(int index, E element): Replaces the element at the specified position.
  • add(int index, E element): Inserts an element at a specific position.
  • remove(int index): Removes the element at a specific position.
  • indexOf(Object o): Returns the first position of an element.

Below we can see an example of who is defined in Java:

Code example 
                        

import java.util.ArrayList;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        // Definir una lista de tipo String
        List<String> lista = new ArrayList<>();

        // Añadir elementos a la lista
        lista.add("Elemento 1");
        lista.add("Elemento 2");
        lista.add("Elemento 3");

        // Imprimir los elementos de la lista
        for (String elemento : lista) {
            System.out.println(elemento);
        }
    }
}

A collection that does not allow duplicate elements. It does not guarantee a specific order.

Base interface: Inherits from Collection.

Main implementations:

  • HashSet: Based on a hash table, does not guarantee order. Fast search and add operations.
  • LinkedHashSet: Maintains insertion order.
  • TreeSet: Based on a red-black tree, keeps elements ordered by their value.

Key methods:

  • put(K key, V value): Associates a key with a value.
  • get(Object key): Returns the value associated with the key.
  • remove(Object key): Removes the key-value pair.
  • containsKey(Object key): Checks if a key is present.
Code example 
                        

import java.util.HashSet;
import java.util.Set;

public class Main {
    public static void main(String[] args) {
        Set<String> conjunto = new HashSet<>();

        conjunto.add("Elemento 1");
        conjunto.add("Elemento 2");
        conjunto.add("Elemento 3");
        conjunto.add("Elemento 1");

        for (String elemento : conjunto) {
            System.out.println(elemento);
        }
    }
}

It does not inherit from Collection, but it is part of the Java Collections Framework. It stores key-value pairs, where each key is associated with exactly one value, and it does not allow duplicate keys.

Base Interface: It is a standalone interface within java.util.

Main Implementations:

  • HashMap: Based on a hash table, it does not guarantee order.
  • LinkedHashMap: Maintains the insertion order.
  • TreeMap: Keeps the keys ordered using a red-black tree.

Key Methods:

  • put(K key, V value): Associates a key with a value.
  • get(Object key): Returns the value associated with a key.
  • remove(Object key): Removes the key-value pair.
  • containsKey(Object key): Checks if a key is present.
  • keySet(): Returns a Set with all the keys.
  • values(): Returns a collection of all the values.
Code example 
                        

import java.util.HashMap;
import java.util.Map;

public class Main {
    public static void main(String[] args) {
        Map<String, Integer> mapa = new HashMap<>();

        mapa.put("Elemento 1", 1);
        mapa.put("Elemento 2", 2);
        mapa.put("Elemento 3", 3);

        for (Map.Entry<String, Integer> entrada : mapa.entrySet()) {
            System.out.println("Clave: " + entrada.getKey() + ", Valor: " + entrada.getValue());
        }
    }
}
